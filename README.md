# SSR 结合 阿里云 FC(function computed) 示例

本示例将我们的[egg-react-ssr](https://github.com/ykfe/egg-react-ssr)项目结合阿里云serverless服务[FC](https://help.aliyun.com/document_detail/52895.html?spm=a2c4g.11186623.6.541.45c9368bqWeNxZ)

使用之前确保电脑已经安装好docker，并且全局安装好[@alicloud/fun](https://github.com/alibaba/funcraft/blob/master/docs/usage/installation-zh.md)

```
$ npm i
$ npm run build
$ npm start
```
